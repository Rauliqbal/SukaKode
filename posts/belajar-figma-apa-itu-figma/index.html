Apa Itu Figma ?

Figma adalah Tools untuk membuat sebuah User Interface yang biasa digunakan oleh seorang UI/UX Designer. Tidak hanya untuk membuat UI saja, tapi juga bisa membuat Prototype.

Figma bisa digunakan di operasi sistem apapun, dikarenakan Figma dijalankan melalui browser, anda tinggal mengunjungi website Figma. Namun Figma juga bisa di install tetapi hanya support untuk Windows dan Mac OS.
